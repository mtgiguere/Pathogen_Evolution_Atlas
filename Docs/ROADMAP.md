# Project Roadmap

This document describes where the project is heading
and how it will evolve over time.

---

## Current stage (v0)

Implemented:
- GenBank ingestion
- Canonical genome normalization
- Mutation detection
- Gene mapping
- Explainable risk scoring
- Automated testing, linting, and security scanning

## Guiding principle

will always be prioritized over black-box prediction.
